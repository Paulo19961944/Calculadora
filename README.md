# Calculadora


Projeto feito pelo Hora de Codar com algumas melhorias. Pois não tinha a integração com o teclado, sendo que nesse modelo já tem a integração com o teclado



# Melhorias a serem feitas

1 -> Aparecer o valor na tela abaixo que é maior e sem o calculo também

2 -> Essa calculadora tem também que pressionar o botao de igual primeiro, porque elá não tem raciocionio lógico de mudar a operação

3 -> Pode adicionar uma calculadora cientifica com histórico




# Modo de Usar

1 -> Pressione o botão e faça o calculo

2 -> Caso precise mudar de operação lembre-se de dar o sinal de igual antes de mudar de operação

3 -> O botão CE apaga os numeros de baixo

4 -> O botão C apaga toudo

5 -> O Botão DEL apaga o número anterior.
